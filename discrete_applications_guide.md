## 四大应用模块

### 1. 🐟 种群动态模型（Ricker模型）

**应用场景**：模拟鱼类等生物种群的动态变化

**模型公式**：
```
N(t+1) = N(t) · exp(r(1 - N(t)/K))
```

**参数说明**：
- `r` (内禀增长率)：0.1 - 3.5，控制种群增长速度
- `K` (承载力)：环境能承载的最大种群数量
- `N₀` (初始种群)：起始种群数量

**功能**：
- 点击"运行模拟"：查看种群时间演化和相图
- 点击"分岔图"：观察随参数r变化，种群从稳定→周期→混沌的转变

**现象观察**：
- r < 2：种群平稳收敛到平衡点
- r ≈ 2-2.7：出现周期振荡
- r > 2.7：进入混沌状态

---

### 2. 🦠 传染病模型（离散SIR模型）

**应用场景**：模拟传染病在人群中的传播过程

**模型方程**：
```
S(t+1) = S(t) - β·S(t)·I(t)
I(t+1) = I(t) + β·S(t)·I(t) - γ·I(t)
R(t+1) = R(t) + γ·I(t)
```

**参数说明**：
- `β` (传染率)：0.0001 - 0.001，每次接触传染的概率
- `γ` (康复率)：0.01 - 0.3，每日康复比例
- **基本再生数 R₀ = β·N/γ**：决定疫情是否爆发

**功能**：
- 运行模拟：观察S-I-R三类人群的时间演化
- 参数扫描：分析不同传染率下的疫情峰值

**关键指标**：
- R₀ > 1：疫情爆发
- R₀ < 1：疫情自然消退
- 峰值感染人数、侵袭率、疫情持续时间

---

### 3. 💹 经济周期模型（蛛网模型）

**应用场景**：描述商品价格和产量的周期性波动

**模型方程**：
```
供给：Q_s(t) = α + β·P(t-1)
需求：Q_d(t) = γ - δ·P(t)
均衡：Q_s(t) = Q_d(t)
```

**参数说明**：
- `β` (供给斜率)：生产者对价格的反应速度
- `δ` (需求斜率)：消费者对价格的敏感度
- **供需比 β/δ** 决定市场稳定性

**功能**：
- 运行模拟：查看价格-产量的时间演化
- 蛛网图：经典的供需曲线蛛网可视化

**市场状态**：
- β/δ < 1：价格收敛，市场稳定
- β/δ = 1：持续振荡
- β/δ > 1：价格发散，市场不稳定

---

### 4. 🦁 捕食者-被捕食者模型（离散Lotka-Volterra）

**应用场景**：描述生态系统中捕食者与猎物的相互作用

**模型方程**：
```
x(t+1) = x(t) + r·x(t)(1 - x(t)) - α·x(t)·y(t)
y(t+1) = y(t) + β·x(t)·y(t) - m·y(t)
```

**参数说明**：
- `r` (猎物增长率)：0.1 - 2.0
- `α` (捕食率)：0.1 - 2.0，捕食效率
- `β` (转化效率)：0.1 - 1.5，食物转化为繁殖的效率
- `m` (捕食者死亡率)：0.1 - 1.0

**功能**：
- 运行模拟：观察两个种群的时间演化
- 相图：显示多条初始条件下的相空间轨迹

**生态现象**：
- 种群周期性波动（捕食者滞后于猎物）
- 平衡点的存在性和稳定性
- 初始条件对系统演化的影响
